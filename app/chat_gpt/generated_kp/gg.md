# **Проект: ГастроExpress**

## **План работы**

### **Краткое описание проекта:**
Разработка мобильного приложения для доставки еды с выбором ресторанов, меню, корзиной, онлайн-оплатой и отслеживанием заказа. Технологии: React Native для фронтенда, Node.js для бэкенда, MongoDB для базы данных. Дополнительно — система рейтингов, push-уведомления и геолокация.

### **Этап 1: Frontend (React Native)**
| **Задача** | **Детализация** |
|------------|-----------------|
| Архитектура и базовая навигация | Настройка проекта на TypeScript, React Native, React Navigation; структура папок, управление состоянием (Redux/Context/Tako), базовые стили и темизация. |
| Экран выбора ресторанов и фильтры | Реализация списка ресторанов с фильтрами по кухне, рейтингу, времени доставки, дистанции; интеграция геолокации для показа nearby. |
| Экран меню ресторана и карточки блюд | Карточки блюд с вариантами порций, добавление в корзину, фильтры по аллергенам, фото и цены; поддержка нескольких вариантов ингредиентов. |
| Корзина и оформление заказа | Мобильная корзина с подсчётом сумм, промокоды, расчет доставки, выбор способа оплаты, оформление заказа. |
| Экран оплаты и подтверждения | Интеграция онлайн-оплаты, обработка статуса транзакции, безопасное хранение данных платежа согласно требованиям. |
| Экран отслеживания заказа | Реалтайм-обновления статуса заказа, карта маршрута, уведомления о смене статуса. |
| Push-уведомления и локальные уведомления | Обработка входящих уведомлений о статусе заказа, настройка тем и звуков. |

### **Этап 2: Backend (Node.js)**
| **Задача** | **Детализация** |
|------------|-----------------|
| Аутентификация и пользователи | Регистрация/вход, JWT-токены, поддержка соцсетей, роли (пользователь, владелец ресторана, админ). |
| Каталог ресторанов и меню | Модели ресторанов, категорий, меню и вариантов блюд; CRUD-операции и кеширование популярных позиций. |
| Заказы, корзины и оплаты | Модели корзины, создание и статус заказов, расчёт цен, интеграция с платежным шлюзом, обработка webhook-уведомлений. |
| Геолокация и поиск | Поиск ближайших ресторанов, радиус доставки, сортировка по времени доставки и рейтингу. |
| Рейтинги и отзывы | Система рейтингов, хранение отзывов, модерация и отображение среднего рейтинга. |
| Push-уведомления и события | Сервис уведомлений и WebSocket/Push для статусов заказа и промо-акций. |

### **Этап 3: Дизайн**
| **Задача** | **Детализация** |
|------------|-----------------|
| UI-кит и дизайн-система | Разработка цветовой палитры, типографики, набор компонентов (кнопки, карточки, формы), документация по стилю. |
| Прототипирование и экранные макеты | Создание макетов ключевых экранов, интерактивные прототипы для проверки UX. |
| Анимации и взаимодействие | Плавные переходы, micro-interactions, анимации добавления в корзину и обновления статусов. |
| Адаптивность и accessibility | Поддержка разных размеров экранов, контрастность, доступность (ARIA/Accessibility-Label). |

### **Этап 4: Деплой и тестирование**
| **Задача** | **Детализация** |
|------------|-----------------|
| CI/CD и окружения | Настройка сборки, автоматизированные тесты, создание окружений staging и productie, секреты и переменные окружения. |
| QA и тестирование | Ручное и регрессионное тестирование ключевых сценариев, тесты производительности. |
| End-to-end тесты | Автотесты пользовательских сценариев ( Detox/Cypress для мобильной части, при необходимости интеграционные тесты API). |
| Подготовка к релизу | Подготовка адаптеров под App Store и Google Play, требования безопасности и приватности, сбор скриншотов и описаний. |
| Мониторинг и поддержка | Настройка мониторинга производительности, логирования, алертов и плана поддержки после релиза. |
